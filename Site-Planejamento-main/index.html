Servidor de Inteligência (Flask).

Uma poderosa API em Python gerencia as requisições, armazena logs silenciosamente e funciona como o cérebro central do motor de detecção.

Segurança no nível do código.

O FraudGuard atua como um "vigia", ele pode monitorar tudo o que um usuário faz no site em que ele está interligado

Dashboard de Monitoramento.

Consome os logs consolidados em JSON sob demanda, para renderizar mapas, gráficos e tabelas de IPs suspeitos. A fim de proporcionar um serviço importante só que de forma intuitiva. Ou seja: Juntamos a eficiência com a compreensão

Fluxo de dados.
Contínuo e Offline.

1

Interação e Registro

O FraudGuard consulta os dados da navegação de diversos usuários (bot ou pessoa) e os envia para o servidor.

2

Persistência

O Servidor Flask armazena esse evento silenciosamente no arquivo access_logs.json.

3

Análise On-Demand

Ao exibir o Dashboard, o log_analyzer.py processa os IPs e aplica as 8 Regras de Detecção para determinar o risco que cada usuário oferece ao site.

4

Visualização

A interface recebe os dados processados em JSON e projeta visualmente as ameaças detectadas para futuras análises.

Inteligência focada no que importa.

!

Rate Limit +30

Muitas requisições em poucos segundos indicando potenciais ataques DDoS ou Scripting.

!

Path Scan +25

Tentativa de acessar caminhos sensíveis expostos, como /.env, /admin e /config.

!

Error Flood +20

IP gerando muitos erros 404 (Não Encontrado) ou 403 (Proibido) sucessivamente.

!

Suspicious UA +15

Uso de ferramentas automatizadas de hacking detectadas (sqlmap, nmap, python-requests).

!

Off-hours +15

Tráfego anormal restrito de madrugada (ex: 00h às 05h), sugerindo automação ou bots.

!

UA Rotation +20

O mesmo IP alterna de User-Agent iterativamente para tentar burlar verificações simples.

!

API Brute Force +25

Excesso de requisições POST nas telas de acesso para fins de forçamento de credenciais.

!

Missing Referrer +20

Acesso direto a páginas sensíveis (como /payment) contornando totalmente a jornada normal.

Visibilidade completa. Em tempo real.
